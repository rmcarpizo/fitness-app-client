# FitForge 💪

A full-stack fitness workout tracker built with Vue 3 and Node.js.

## Live Demo
🔗 [FitForge](https://fitness-app-client-beta.vercel.app)

## Features
- User registration and login with JWT authentication
- Add, edit, delete, and complete workouts
- Track total workouts, completed sessions, and minutes logged
- Protected routes — only logged-in users can access workouts

## Tech Stack
**Frontend:** Vue 3, Vite, Pinia, Vue Router, Bootstrap 5, Axios  
**Backend:** Node.js, Express, MongoDB Atlas, Mongoose, JWT, Bcrypt  
**Deployed on:** Render (backend) · Vercel (frontend)

## Installation

### Frontend
```bash
cd fitnessApp-client
npm install
npm run dev
```
